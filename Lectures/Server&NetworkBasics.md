## Server and Network Basics

- **Server Side:** Business logic and data operations are stored in the database.  
- **Frontend:** Displays UI elements (like buttons) for user interaction.

### Network Communication
- Computers connect via **IP addresses** (e.g., `127.0.0.1`).  
- Example: `localhost:3000` or `127.0.0.1:3000` â€” where **3000** is the **port number**.  
- `https://` indicates a **secure** connection.  

### Common Ports
- **PostgreSQL:** Port `5432` (default, can be changed).  
- **Redis:** Runs on separate ports.  

Using **standard conventions** helps avoid confusion.
